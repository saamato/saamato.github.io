Forbes 404 Page

Role: UX Designer
Duration: ~1 month

Overview

I was tasked with updating the 404 page for the account center in Forbes. We wanted to provide users with more entry points into Forbes while also cleaning up the UI for the error warning.

Why are you yelling at me?

Problem

Simply put, the 404 page in the account center was outrageous. When a user hit an error or a page that did not exist, the 404 page would essentially yell at the user, giving them no useful information for what went wrong.

Solution

We wanted to give the user some kind of message that let them know someone or something made a mistake. We also wanted to make it uniquely Forbes. We decided to anchor the page with a small error code and a quote from Malcolm Forbes. But we also wanted to give the user other options than just returning to Forbes.com. We included four customizable blocks below the error to allow for entry points into the Forbes ecosystem.

Results

A brand new 404 page with much less yelling
